Dead Mobiles and their Solutions ( English )

There can be various reasons for mobile phone to be dead which we can easily check and repair. Tools for this work are necessary like multimeter and DC power supply and hot air gun for better use of tools. It is important to read the classes many mobile phones are dead only because of battery problem first check by installing a good battery or turn it on with DC power supply. 25% of mobile phones are dead due to the situation.

The fault of the battery terminals is damaged or dry solid, replace the battery terminal if it is damaged, you often get dead mobiles due to this fault. Check the PCB and battery terminals, if carbon is visible, clean it with CTC.

If the power circuit or power button is damaged, the mobile phone goes dead. Check the power button with a multimeter. If it is damaged, replace the power button and check with the help of the circuit diagram. If the circuit is broken, install a jumper wire. A good soldering iron should be used for this task.

If the strap of a flip mobile is damaged, it becomes dead, in such a case, replace the strap.

You can check with the help of regular power supply, if the mobile is dead due to software, then install new software in it. Installing new software in the mobile is called flashing .

When you wash the board with CTC , always protect the LCD from it otherwise marks are formed inside the LCD.

In case of dry solid in Power IC or CPU section, the mobile phone becomes dead, in such a case, repair the CPU with the help of a hot air gun by applying a small paste to the CPU. It must be used properly otherwise the mobile phone can be dead forever and it can be not only difficult but impossible to repair it, so repair it very carefully.

How to Check Dead Set

Check by replacing the battery or give the battery a shot from a 12 volt supply or turn on the set with the help of a power supply - to turn on the mobile the battery must be three point seven volts.

Check the temperature and BSI terminal, if it is bad, replace it.

Reload all CPU section If after resoldering still no power then replace power IC

Give flash IC a light heat or replace it If flash IC is removed then mobile software is out Done Inspect the board in detail if any IC is cracked or burnt then replace it

Repairing Instructions

If any component on the PCB board is not working properly or is not working, then to repair it, Google the PCB diagram of the same model and the diagram of the component to be repaired. Where are the cap points passing through and then check with a multimeter if there is no reading then they are putting jumpers.

To repair any set, the most important thing is to open it to see how to do it. Watch the video from YouTube.

Sometimes if you do not understand any fault, then watch it from YouTube so that our mobile can be saved from further damage.
