SMD (Surface Mounting Devices) in English

Following are the SMD components,

  • Non-Polar Capacitor.
  • Polar Capacitor.
  • Resistance.
  • Front leg IC.
  • Bubble IC.
  • Diode.
  • Serial Capacitor.
  • Serial Resistance.
  • Ground.
  • Transistor.

Non-Polar Capacitor :

A non-polar capacitor is only one color on the PCB board and that color is light brown, but it can be of different sizes, it does not have positive and negative terminals, and it does not have the ability to store voltage within itself.

Polar Capacitor :

The basic (4) colors of a polar capacitor are black, white, blue and green. Only these four colors are combined to form further colors, mostly yellow and orange across the board.

Resistance :

They are black on top of the board and have white solder on both sides.
The second type is used in most electronics. There are different colored lines on the top. Each line has a specific value.
( -) and (+) does not happen, you can check it with a multimeter, set the multimeter to ohms. It is never shorted but opens when it is damaged or its value is full and its value must be removed from the circuit.

Code of SMD Resistance
3rd digit of Code shows the Number of Zeros.
For Example,
Code 330 = 33 ohm
Code 221 = 220 ohm
Code 683 = 6800 ohm

Polar Capacitor :

Resistance :

یہ بورڈ کے اوپر کالے رنگ میں ہوتے ہیں اور دونوں اطراف میں سفید سولڈر لگا ہوتا ہے
